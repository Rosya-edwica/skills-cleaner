# skills-cleaner

## План работы
1. Регулярка 
2. Минус навыки 
3. 100% Повторения без учета регистра 
4. Грамматика
5. Инфинитив
6. FuzzyVuzzy


## Текущая реализация:

На данный момент программа может:
1. Очищать навыки от стоп-слов
2. Очищать названия навыков от ненужных символов
3. Переводить навыки на английский язык
